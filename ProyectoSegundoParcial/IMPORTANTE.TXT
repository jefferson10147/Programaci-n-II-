Profesor, esperamos se encuentre muy bien, estas son algunas consideraciones a la hora de correr nuestro proyecto: 

		* Antes de correr el proyecto debe por si acaso modificar el archivo classDB.php
		ya que allí se encuentra la configuración sql, debe cambiar la contraseña para que se pueda
		conectar a su gestor de bases de datos, el campo a modificar es:
			 
			private static $dbPassword = 'admin123';

		* Para ingresar al sistema como usuario normal puede usar el perfil:
			
			->usuario: mary
			->clave: clave

		* A la hora de registrar un Boleto como usuario normal, el nombre y la fecha 
		del evento deben ser válidas de un evento previamente registrado, si desea 
		registrar un boleto puede usar la siguiente información:

			->serial: 213308 (cualquier string sería válido)
			->nombre del evento: of mosters and men
			->fecha del evento: 27/10/2020
			->localización: (cualquiera)


		* Para ingresar al sistema como usuario administrador, puede usar el perfil:
	
			->usuario: "root@jefferson"
			->clave: "root"
